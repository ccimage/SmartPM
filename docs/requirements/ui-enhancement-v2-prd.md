# SmartPM UI 增强 v2 需求文档

**版本**：v1.0  
**日期**：2026-05-12  
**范围**：主界面布局重构、项目图标与颜色、项目列表改版、任务字段美化、任务评论与活动日志

---

## 一、背景与目标

在完成 M1–M4（主题换肤、头像、视觉改版、富文本）之后，本轮聚焦以下方向：

- 简化主界面布局，去掉侧边栏，降低视觉噪音。
- 为项目增加图标与颜色标识，提升多项目场景下的辨识度。
- 项目列表支持大图/列表两种浏览模式，适应不同使用习惯。
- 任务字段增加图标，标签彩色化，提升信息密度与可读性。
- 任务增加评论与操作日志，支持团队协作沟通。

---

## 二、详细需求

### 2.1 主界面布局重构

**需求描述**

去掉左侧边栏，改为纯上下两栏布局：顶栏 + 内容区。

顶栏结构（从左到右）：
- 左侧：SmartPM logo 图标 + "SmartPM" 文字品牌区
- 中间：面包屑导航（工作区名称 / 项目名称，可点击跳转）
- 右侧：用户头像 + 用户名 + 下拉菜单指示器（▾）

下拉菜单内容：
- 个人资料（跳转 `/settings/profile`）
- 外观设置（跳转 `/settings/appearance`）
- 分割线
- 退出登录

下拉菜单交互：
- 点击用户区域展开/收起
- 展开/收起带过渡动画（淡入 + 向下滑出，约 150ms）
- 点击菜单外部区域自动关闭（`onClickOutside`）

**业务规则**

- 面包屑根据当前路由动态生成：
  - 无工作区：仅显示"工作区"
  - 有工作区无项目：显示"工作区名称"
  - 有工作区有项目：显示"工作区名称 / 项目名称"
- 面包屑每一段均为可点击链接，点击跳转对应列表页。
- 下拉菜单在移动端同样可用。

**验收标准**

- 登录后主界面无左侧边栏，仅顶栏 + 内容区。
- 顶栏左侧显示 SmartPM logo 和品牌名。
- 面包屑正确反映当前路由层级，各段可点击跳转。
- 用户名右侧有下拉指示器，点击展开菜单，菜单有动画，点击外部关闭。
- 退出登录和设置入口均在下拉菜单中。

### 2.2 项目图标与颜色

**需求描述**

创建和编辑项目时，可以为项目指定图标和颜色。

图标选择：从 Font Awesome 6 Free Coding 分类中选取 50 个图标，包含：
`code`、`code-branch`、`code-commit`、`code-compare`、`code-fork`、`code-merge`、`code-pull-request`、`terminal`、`bug`、`bug-slash`、`database`、`server`、`laptop-code`、`file-code`、`microchip`、`memory`、`hard-drive`、`network-wired`、`sitemap`、`diagram-project`、`cube`、`cubes`、`layer-group`、`puzzle-piece`、`plug`、`hashtag`、`at`、`percent`、`infinity`、`superscript`、`subscript`、`object-group`、`object-ungroup`、`bolt`、`fire`、`rocket`、`star`、`flag`、`bookmark`、`tag`、`tags`、`folder`、`folder-open`、`file`、`file-lines`、`list`、`list-check`、`table-columns`、`chart-bar`、`chart-line`

颜色选择：提供 12 个预设颜色 + 自定义 hex 输入。

图标和颜色在以下位置展示：
- 项目列表卡片（大图标，带颜色背景圆角块）
- 面包屑中的项目名称前（小图标，带颜色）
- 任务看板顶部项目标题区

**业务规则**

- 图标和颜色均为可选字段，未设置时使用默认图标（`code`）和默认颜色（`#4f46e5`）。
- 颜色值保存为 hex 字符串（如 `#4f46e5`）。
- 图标值保存为 Font Awesome 图标名（如 `code`，不含 `fa-` 前缀）。

**验收标准**

- 创建/编辑项目表单中有图标选择器和颜色选择器。
- 图标选择器展示 50 个图标，可滚动选择，选中后高亮。
- 颜色选择器提供 12 个预设色块 + 自定义 hex 输入。
- 保存后项目卡片和面包屑正确展示图标和颜色。

### 2.3 项目列表界面改版

**需求描述**

项目列表页面改版，支持两种浏览模式：

**大图模式（默认）**：
- 卡片网格布局，每行 3–4 个
- 卡片展示：项目图标（大，带颜色背景）、项目名称、描述摘要、成员头像组、任务数量
- 鼠标悬浮时，卡片右上角出现快捷操作区：颜色修改按钮、图标修改按钮、更多菜单（编辑、删除）
- 快捷颜色修改：点击颜色按钮，弹出小色盘浮层，选色后立即保存
- 快捷图标修改：点击图标按钮，弹出图标选择浮层，选择后立即保存

**列表模式**：
- 紧凑行列表，每行显示：图标色块、项目名称、描述、成员数、任务数、创建时间、操作按钮
- 操作按钮：编辑、删除

**模式切换**：
- 顶部右侧有大图/列表切换按钮（图标按钮组）
- 切换偏好保存到 `localStorage`，刷新后保持

**验收标准**

- 项目列表默认大图模式，可切换为列表模式。
- 大图模式下鼠标悬浮显示快捷颜色/图标修改入口。
- 快捷修改颜色/图标后无需刷新，卡片即时更新。
- 列表模式下信息紧凑展示，操作按钮可用。
- 视图模式偏好持久化到 `localStorage`。

### 2.4 任务字段图标化与标签彩色化

**需求描述**

任务创建/编辑界面中，每个字段行左侧增加对应小图标：

| 字段 | 图标 |
|------|------|
| 标题 | `pen-to-square` |
| 负责人 | `user` |
| 截止时间 | `calendar` |
| 优先级 | `flag` |
| 标签 | `tag` |
| 描述 | `align-left` |
| 附件 | `paperclip` |
| 状态 | `circle-dot` |

**标签彩色化**：
- 标签支持颜色属性，保存为 hex 颜色值
- 标签展示为彩色胶囊样式（背景色为标签色的浅色版，文字为深色版）
- 创建标签时可选颜色（12 个预设颜色）
- 任务编辑界面支持快捷添加标签：输入框 + 下拉候选列表，支持创建新标签

**验收标准**

- 任务编辑界面每个字段行左侧有对应图标。
- 标签以彩色胶囊样式展示。
- 可在任务编辑界面快捷添加/移除标签。
- 创建标签时可选颜色，颜色在标签上正确展示。

### 2.5 任务评论与操作日志

**需求描述**

任务详情界面底部增加两个 Tab：

**评论 Tab**：
- 评论列表：头像、用户名、时间、评论内容（支持富文本）
- 评论输入框：富文本编辑器（minimal 工具栏）
- 提交按钮
- 支持删除自己的评论

**操作日志 Tab**：
- 展示该任务的操作历史，例如：
  - "张三 创建了任务"
  - "李四 将状态从「待处理」改为「进行中」"
  - "张三 修改了截止时间"
  - "李四 添加了标签 #bug"
- 每条日志显示：操作人头像、操作描述文字、时间

**业务规则**

- 评论按时间正序排列（最新在下）。
- 操作日志按时间正序排列。
- 评论内容经后端 HTML 白名单清洗后保存。
- 只有评论作者可以删除自己的评论。
- 操作日志由后端在任务 create/update/delete 时自动写入，前端只读。

**验收标准**

- 任务详情界面底部有"评论"和"日志"两个 Tab。
- 可在评论 Tab 发表评论，评论即时显示。
- 可删除自己的评论。
- 日志 Tab 展示该任务的操作历史记录。
